# Система за оценки и коментари на филми (Laravel)

Курсова работа: уеб приложение за **филми**, в което потребителите добавят **оценки (1–10)** и **коментари**. Поддържа **средна оценка**, **търсене по заглавие/жанр**, и **административен панел**.

## Основни функционалности (цел)

-   **Филми**: заглавие, година, жанрове, постер (качен файл)
-   **Ревюта**: числова оценка + текстов коментар + време на добавяне + автор
-   **Средна оценка** за филм
-   **Търсене** по заглавие и жанр
-   **Админ панел**: CRUD за основните обекти + добавяне на админ потребители

## Стартиране (локално)

1. Копирай `.env.example` → `.env` и настрой DB връзката (по подразбиране може да ползваш SQLite `database/database.sqlite`).
2. Генерирай ключ:
    - `php artisan key:generate`
3. Инсталирай зависимости (ако имаш Composer/Node):
    - `composer install`
    - `npm install`
4. Storage link (за качени файлове/постери):
    - `php artisan storage:link`
5. Миграции и seed:
    - `php artisan migrate --seed`
6. Старт:
    - `php artisan serve`
    - (за dev стилове) `npm run dev` **или** `npm run build` (ако няма да разработваш)

Алтернатива (стартира server + queue + vite наведнъж):

-   `composer run dev`
